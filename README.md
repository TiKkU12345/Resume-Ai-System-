# 🎯 AI Resume Shortlisting System  
### Smart Resume Screening & Candidate Ranking using AI

An end-to-end **AI-powered recruitment system** that automates resume shortlisting, reduces manual screening effort, and ranks candidates based on job relevance.

> Designed to help recruiters and HR teams make **faster, unbiased, and data-driven hiring decisions**.

---

## 🚀 Why This Project Exists
Traditional resume screening is:
- Time-consuming  
- Prone to human bias  
- Inefficient at scale  

This system uses **AI + NLP** to intelligently evaluate resumes and shortlist the best candidates in seconds.

---

## ✨ Key Features
- 📄 **Resume Parsing** – Supports PDF & DOCX resumes  
- 🧠 **AI-Powered Matching** – Matches resumes with job descriptions using LLMs  
- 📊 **Candidate Ranking** – Scores and ranks candidates based on relevance  
- 🔐 **Admin Approval Workflow** – Manual control over final shortlisting  
- 🗂 **Centralized Resume Database** – Secure storage with Supabase  
- ⚡ **Real-time UI** – Interactive Streamlit dashboard  

---

## 🧠 How It Works (Flow)
1. Recruiter uploads resumes  
2. Job description is provided  
3. AI extracts skills, experience & keywords  
4. Resume-JD similarity score is generated  
5. Candidates are ranked automatically  
6. Admin reviews & approves shortlisted profiles
